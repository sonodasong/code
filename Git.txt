In this file, I am going to write down some basic usages of Github
-----------------------------------------------------------------------------
1. Clone some code project, check the version and do update
	1.1 to download the project
		git clone "link" 

	1.2 to check the version of the project
		git tag -l
	
	1.3 to make sure that the code is up to date
		git pull
--------------------------------------------------------------------------------
2. Push the local project to Github
	Create a repository on the wesite first to get *.git link
	change to the project directory
	git init
	git status
	git add -A
	git commit -m "messgaes"
	git remote add origin *.git
	git remote -v
	git push
